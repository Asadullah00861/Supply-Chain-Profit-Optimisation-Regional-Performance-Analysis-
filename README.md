# ProfitPulse-Market-Regional-Performance-with-Forecasting

This project focuses on analyzing supply chain profit performance across different markets and regions. It combines exploratory data analysis (EDA), feature engineering, and predictive modeling to uncover profit trends, evaluate year-over-year (YoY) growth, and forecast future profitability. An interactive Excel dashboard is included to present key insights in a business-friendly format.



## Objectives
- Analyze profit distribution across markets and regions
- Measure year-over-year growth and performance trends
- Identify key drivers of profitability
- Build predictive models to forecast regional profit
- Present insights through an interactive dashboard

## Files
- **Work_File.ipynb** — Primary analysis notebook  
  - Loads and preprocesses raw data  
  - Performs exploratory data analysis (EDA)  
  - Trains machine learning models  
  - Key objects: `df`, `predict_profit_region`, `tree_model1`, `lr_model1`
- **incom2024_delay_example_dataset.csv** — Original raw dataset  
- **incom2024_delay_variable_description.csv** — Data dictionary and variable descriptions  
- **dashboard.xlsx** — Interactive dashboard presenting KPIs, YoY growth, market and regional profit analysis, and forecasted insights
